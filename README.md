# 🔐 Gerador de Senhas Seguras

Um aplicativo web simples e responsivo que permite aos usuários gerarem senhas aleatórias fortes com base em critérios customizados (comprimento, letras, números e símbolos).

## 🚀 Tecnologias Utilizadas
- **HTML5** para estruturação semântica.
- **CSS3** com variáveis e layout responsivo (Flexbox).
- **JavaScript (ES6+)** para lógica de criptografia simulada e manipulação do DOM.

## 🛠️ Funcionalidades
- Escolha de comprimento da senha (6 a 25 caracteres).
- Filtros por Maiúsculas, Minúsculas, Números e Símbolos.
- Botão "Copiar" integrado utilizando a Clipboard API do navegador.
